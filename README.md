# Zhur

## What is Zhur?

Zhur [ êur], named after [the soup](https://en.wikipedia.org/wiki/West_Slavic_fermented_cereal_soups#Poland), is a comprehensive platform for writing simple, small-scale HTTP-driven web apps with WebAssembly, utilizing Kevin Hoffman's waPC protocol.

## Features (WIP):

- HTTP request handling
- Built-in key-value store for quick data access
- App management portal
- Use any language with WebAssembly support to write your apps
